# dotfiles


```bash
Install [Homebrew](http://brew.sh/)
brew install git
setup 1password
setup ssh for Github https://help.github.com/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent/
git clone git@github.com:Andersos/dotfiles.git
cd dotfiles
source bootstrap.sh
chmod +x ./brew.sh
./brew.sh
Activate filevault  
```
